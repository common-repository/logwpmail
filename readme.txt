== Description ==
Log all outgoing mail for testing or recordkeeping.

Log files are stored in the log/ subdir if you choose to remove the files manually.  Log files are not cached or backed up so delete at your own risk.

There is a new file initially generated for each day if an email is generated using wp_mail or any other email plugin.

Files can be Purged by checking the "Purge files every...." checkbox.

== Installation ==
1. After downloading the ZIP file,
2. Log in to the administrator panel.
3. Go to Plugins Add > New > Upload.
4. Click "Choose file" ("Browse") and select the downloaded zip file.
5. Click "Install Now" button.
6. Click "Activate Plugin" button for activating the plugin.
7. Click 'Log WP_Mail' in the dashboard menu

== Frequently Asked Questions ==
= How do I use it =
1.  Click 'Log WP_Mail' in the dashboard menu
2. Send out a test email
3. Reload the Log WP_Mail settings page to see your log entries
